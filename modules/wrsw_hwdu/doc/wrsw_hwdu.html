hwdu_wishbone_slave

WR Switch Hardware Debugging Unit

The module is used for reading the value of registers from inside of WR Switch. Usefull for debugging during development.

  • ADR [read/write]: Address of the register
    Which register (among those connected to HWDU) will be read
  • RD_ERR [read-only]: Read error
    read 1: read error, provided address is out of range
    read 0: read done successfully
  • RD_EN [read/write]: Read register value
    write 1: read the content
    write 0: no effect
    read 1: reading in progress
    read 0: reading done, register value available
