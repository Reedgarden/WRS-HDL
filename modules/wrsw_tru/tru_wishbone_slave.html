3. Register description

3.1. TRU Global Control Register

HW prefix: tru_gcr
HW address: 0x0
C prefix: GCR
C offset: 0x0

Control register containing global (port-independent) settings of the TRU.

31 30 29 28 27 26 25 24
RX_FRAME_RESET[23:16]
23 22 21 20 19 18 17 16
RX_FRAME_RESET[15:8]
15 14 13 12 11 10 9 8
RX_FRAME_RESET[7:0]
7 6 5 4 3 2 1 0
- - - - - - TRU_BANK G_ENA
  • G_ENA [read/write]: TRU Global Enable
    Global TRU enable bit. Overrides all port settings.
    0: RTU is disabled. All packets are dropped.
    1: RTU is enabled.
  • TRU_BANK [write-only]: Swap TRU TAB bank
    write 1: swaps the active bank of the VLAN TAB (writing to TRU TAB affects
    inactive bank, only swapping the banks causes the written data to be activated
    write 0: no effect
  • RX_FRAME_RESET [read/write]: Rx Frame Reset
    Resets information about filtered frames received on
    a port

3.5. Link Aggregation Control Register

HW prefix: tru_lacr
HW address: 0x4
C prefix: LACR
C offset: 0x10

Enables configuration of Link Aggregation distribution functions for each kind of traffic.
Available functions:
0: based on pclass detected by Packet Filter (need proper pFilter config)
1: based on destination MAC address (bits 6 and 7)
2: based on source MAC address (bits 6 and 7)

- - - - AGG_DF_HP_ID[3:0]
  • AGG_DF_HP_ID [read/write]: HP traffic Distribution Function ID
    ID of Aggregation Distribution Function for HP traffic (value of 0 recommended, requires proper pFilter config)
  • AGG_DF_BR_ID [read/write]: Broadcast Distribution Function ID
    ID of Aggregation Distribution Function for broadcast traffic (value of 2 recommended)
  • AGG_DF_UN_ID [read/write]: Unicast Distribution Function ID
    ID of Aggregation Distribution Function for unicast traffic

3.6. Transition Control General Register

HW prefix: tru_tcgr
HW address: 0x5
C prefix: TCGR
C offset: 0x14

Defines transition mode/configuration for slow port reconfiguration - decides
when two swap banks such that HP packets are not lost.

31 30 29 28 27 26 25 24
- - - - - - - -
23 22 21 20 19 18 17 16
- - - - - - - -
15 14 13 12 11 10 9 8
TRANS_PRIO_MODE TRANS_PRIO[2:0] - TRANS_RX_ID[2:0]
7 6 5 4 3 2 1 0
- TRANS_MODE[2:0] - - TRANS_CLEAR TRANS_ENA
  • TRANS_ENA [read/write]: Transition Enabled
    Enables/disables transition
  • TRANS_CLEAR [read/write]: Transition Clear
    Writing 1 clears >Transition Finished< bit so that new transition can be
    performed. No new transition will be started until >Transition Finished< is cleared
  • TRANS_MODE [read/write]: Transition Mode
    Selected Transitin Mode for port re-config
  • TRANS_RX_ID [read/write]: Rx Detected Frame ID
    Base transition on detection of the frame which is parsed into provided CLASS ID
  • TRANS_PRIO [read/write]: Priority
    Indicates at which traffic priority the transition attempts not to loose
    frames
  • TRANS_PRIO_MODE [read/write]: Priority Mode
    Specifies whether
    - 0: use indication of HP packet from RTU (fast match)
    - 1: use the priority specified in TRANS_PRIO register
    to count packets during transition
